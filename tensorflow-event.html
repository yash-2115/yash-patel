Tensorflow Dev Summit 2019 Viewing Party

Event information

  • Community: GDG (Google Developers Group) Cloud Ahmedabad
  • Event date: 16 March, 2019
  • Event URL: Tensorflow Dev Summit 2019 Viewing Party

TensorFlow Dev Summit brings together a diverse mix of machine learning users from around the world for two days of highly technical talks, demos, and conversation with the TensorFlow team and community. It is no question that machine learning is one of the hottest and one of the most important technology topics for developers and we think you can use this as an opportunity to build local machine learning community or engage with your local machine learning users!
